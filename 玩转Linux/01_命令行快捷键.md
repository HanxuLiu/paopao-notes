# 命令行快捷键

## 光标移动到首尾

- 光标移动到行首：`Ctrl + a`
- 光标移动到行首：`Ctrl + e`

![ctrl_a_e](https://cdn.jsdelivr.net/gh/HanxuLiu/CDN1/img/2023/202306061423573.gif)

## 基于光标位置删除

- 删除光标到行首的内容（不含光标）：`Ctrl + u`
- 删除光标到行尾的内容（包含光标）：`Ctrl + k`

![ctrl_u](https://cdn.jsdelivr.net/gh/HanxuLiu/CDN1/img/2023/202306061423405.gif)
